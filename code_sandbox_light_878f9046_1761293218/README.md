## üßë‚Äçüíª Local Preview & Development

The project is a static site, so you can preview updates locally with any lightweight server. Pick whichever option matches your tooling:

### Option 1: VS Code Live Server (Recommended)
1. Install the **Live Server** extension.
2. Right-click `index.html` and choose **Open with Live Server**.
3. Your default browser opens `http://127.0.0.1:5500/` (or similar) and auto-reloads on save.

### Option 2: Node.js `http-server`
1. Install Node.js, then run `npm install --global http-server`.
2. From the project root, start the server with `http-server -c-1`.
3. Visit `http://localhost:8080` to browse every page.

### Option 3: Python `http.server`
1. With Python 3 installed, run `python -m http.server 8080` in the project root.
2. Navigate to `http://localhost:8080`.

> Tip: Keep `index.html` open in a second tab while editing other pages so you can jump through the site quickly.

## üöÄ Deployment Instructions

### **Option 1: Netlify (Recommended - Easiest)**

1. **Create Netlify Account:**
   - Go to [netlify.com](https://netlify.com)
   - Sign up for free account

2. **Deploy Website:**
   - Click "Add new site" ‚Üí "Deploy manually"
   - Drag and drop all project files
   - Or connect to Git repository for automatic deployment

3. **Connect Custom Domain:**
   - Go to Site settings ‚Üí Domain management
   - Add custom domain: `mybanaras.com`
   - Update DNS records as instructed:
     - Add A record: `75.2.60.5`
     - Add CNAME: `www` ‚Üí your-site-name.netlify.app
   - SSL certificate automatically generated

4. **Done!** Your website is live at `https://mybanaras.com`

**Time Required:** 5-10 minutes (no coding needed)

---

### **Option 2: Vercel**

1. **Create Vercel Account:**
   - Go to [vercel.com](https://vercel.com)
   - Sign up for free

2. **Import Project:**
   - Click "New Project"
   - Import from Git or upload files
   - Deploy automatically

3. **Add Custom Domain:**
   - Go to Project Settings ‚Üí Domains
   - Add `mybanaras.com`
   - Update DNS records as instructed

**Time Required:** 5-10 minutes

---

### **Option 3: GitHub Pages**

1. **Create GitHub Repository:**
   - Go to [github.com](https://github.com)
   - Create new repository: `mybanaras`

2. **Upload Files:**
   - Upload all project files to repository
   - Go to Settings ‚Üí Pages
   - Select branch: `main`, folder: `/ (root)`
   - Click Save

3. **Custom Domain:**
   - Add CNAME file with content: `mybanaras.com`
   - Update DNS: CNAME `www` ‚Üí `username.github.io`

**Time Required:** 10-15 minutes


### **Update Contact Information:**
Edit `contact.html` with your actual:
- Email addresses
- Phone numbers
- Physical address

### **Add Google Analytics:**
Add before `</head>` tag in all HTML files:
```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```

### **Add Google AdSense:**
Replace `.ad-placeholder` content with AdSense code:
```html
<div class="ad-placeholder">
  <!-- Google AdSense code here -->
  <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
  <ins class="adsbygoogle" ...></ins>
</div>
```

### **Update Social Media Links:**
Edit footer in all HTML files:
```html
<div class="social-links">
  <a href="YOUR_FACEBOOK_URL">...</a>
  <a href="YOUR_INSTAGRAM_URL">...</a>
  <a href="YOUR_YOUTUBE_URL">...</a>
```
